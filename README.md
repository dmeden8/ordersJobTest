### Startup: ###
#### 1. run src/main/resources/sql/orders-role-DDL.sql to create DB role ####
#### 2. run src/main/resources/sql/orders-db-DDL.sql to create DB ####
#### 3. set 'ddl-auto: create' in src/main/resources/application.yml -> Hiberante will create tables and constraints ####
#### 4. in project root directory run: 'mvnw clean spring-boot:run' ####
#### 5. set 'ddl-auto: validate' in src/main/resources/application.yml -> Tables exist, remove create ####
